# 位运算

## 1、按位与（&）
+ 全部为1，则为1 
## 2、按位或（|）
+ 至少一个1，则为1；全部为0，则为0
## 3、按位异或（^）
* 全部相同为0（全是1/全是0），不同为1
## 4、按位取反（~）
+ 按位取反
## 5、按位左移（<<）相当于n*2
- 左移动指定位数，右补0
## 6、按位右移（>>）相等于n/2
- 右移动指定位数，左补0
